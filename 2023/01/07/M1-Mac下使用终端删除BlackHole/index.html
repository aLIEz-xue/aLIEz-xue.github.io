使用終端在Mac上卸載BlackHole

除了從您的計算機手動卸載音頻驅動程序外，您還可以使用終端Terminal來執行此操作。 這些步驟很容易遵循。 只需在下面查看：

  1. 轉到 Mac 計算機中的系統設置。
  2. 在該系統設置上，確保關閉並退出音頻設備。 其中包括關閉聲音和關閉音頻 MIDI 設置。
  3. 現在，打開 *終端*.
  4. 現在，您必須訪問找到上述 BlackHole 音頻驅動程序的目錄，鍵入命令： ***\*cd /Library/Audio/Plug-Ins/HAL\****
  5. 確保音頻驅動程序BlackHole.driver位於該位置。 要在 Mac 上卸載 BlackHole，您必須刪除上述驅動程序。 要刪除’BlackHoleXch.driver’，可在終端上鍵入以下命令：
  • 刪除舊驅動程序： sudo rm -rf BlackHole.driver
  • 刪除16CH版本： sudo rm -rf BlackHole16ch.driver
  • 刪除2CH版本： sudo rm –rf BlackHole2ch.driver

  1. 輸入命令： sudo launchctl kickstart -kp system/com.apple.audio.coreaudiod , 重新啟動 Mac 計算機中音頻的主要或核心驅動程序。
  2. 現在你可以 檢查您的音頻 MIDI 設置 或您的系統偏好設置。 確保 BlackHole 的音頻驅動程序不再可用。
